# ✨Java Learning Journey - Bootcamp Generation

Este repositorio contiene mi progreso y proyectos desarrollados durante el bootcamp de **Desarrollador Full Stack Java Jr.** de Generation Chile. Aquí documento mi transición desde la Licenciatura en Literatura Creativa hacia la Ingeniería de Software.

## 🗂️ Estructura del Repositorio

El proyecto está organizado en módulos que cubren desde los fundamentos sintácticos hasta la lógica de programación compleja:

### 1. 🧮 [AdvancedCalculator](./Bucles/AdvancedCalculator)
Una calculadora de consola de alta precisión.
* **Funcionalidades:** Operaciones básicas, potencias (`Math.pow`), raíces (`Math.sqrt`) y manejo de errores (división por cero).
* **Conceptos clave:** Métodos estáticos, tipos de datos `double` y modularización.

### 2. 🔄 [Bucles & Control de Flujo](./Bucles/Bucles)
Ejercicios enfocados en la iteración y toma de decisiones.
* **Contenido:** Implementaciones de `for`, `while` y `do-while`.
* **Proyecto Destacado:** `LoopCommerce.java` (Simulación de inventario y ventas).

### 3. 🎮 [RPG Menu Simulator](./Bucles/RPG)
Un simulador de menús para juegos de rol.
* **Aprendizaje:** Manejo de estructuras `switch` anidadas y lógica de estados del jugador.

### 4. 📝 Fundamentos Iniciales
* **HolaMundo.java:** Mi primer contacto con la sintaxis de Java.
* **MenuVariables.java:** Declaración y tipado de datos.
* **BoletaSimple.java:** Ejercicio práctico de entrada y salida de datos (`Scanner`).

## 🛠️ Tecnologías y Herramientas
* **Lenguaje:** Java 21 (JDK)
* **IDE:** Visual Studio Code (macOS)
* **Control de Versiones:** Git & GitHub
* **Gestión de Terminal:** Zsh (MacBook Air)

## 🧠 Reflexión Académica
Como graduada en **Literatura Creativa**, estudio el código como una forma de arquitectura lingüística para entenderlo como una extensión del lenguaje natural. Este repositorio no solo guarda funciones y bucles, sino que representa el análisis sintáctico aplicado a la resolución de problemas lógicos y tecnológicos.
